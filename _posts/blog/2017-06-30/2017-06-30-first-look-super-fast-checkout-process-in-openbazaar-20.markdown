The wallet talks directly to the Bitcoin network, meaning all transactions are on-chain and there are no servers. **Okay...but what's an SPV wallet?Â ** The most secure way to use Bitcoin is to run a "full node" which means you have the entire copy of the blockchain and you verify all new transactions in new blocks yourself. However, running a full node is inconvenient, requiring a significantÂ amount of disk space and other resources. The average user isn't going to download the entire blockchain.

> That's where **Simplified Payment Verification (SPV)** wallets come in.

The basic idea is that an SPV wallet doesn't verify everything in blocks as accurate, so it doesn't need to download the entire chain. It relies on a trusted node for verification.
